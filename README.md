# Spotify A3 — Análise Preditiva

Projeto de análise e mineração de dados sobre músicas do Spotify, seguindo o processo KDD.

## Dashboard

O dashboard interativo desenvolvido em Streamlit está disponível em:

[spotify-a3-hjqu7qwaksw7h6ze3ykue5.streamlit.app](https://spotify-a3-hjqu7qwaksw7h6ze3ykue5.streamlit.app)

## Estrutura do Projeto

```
spotify-a3/
├── Artigo/                  # Artigo científico do projeto (PDF)
├── Codigo-Transformação/    # Notebook de limpeza e transformação dos dados
├── Codigo-Mineração/        # Notebook de mineração e modelagem preditiva
├── Raw-Dataset/             # Dataset bruto utilizado como entrada
├── app.py                   # Código-fonte do dashboard Streamlit
└── requirements.txt         # Dependências do projeto
```

## Conteúdo das Pastas

- **Artigo/** — Documento completo do trabalho, descrevendo metodologia, análises e resultados.
- **Codigo-Transformação/** — Etapa de pré-processamento: limpeza, normalização e preparação dos dados.
- **Codigo-Mineração/** — Etapa de mineração: aplicação de algoritmos de machine learning e análise dos resultados.
- **Raw-Dataset/** — Dataset original em formato CSV.
